# Installation

Get started with Ignix UI by adding it to your React project. This guide will walk you through the installation process and help you set up everything you need to start building beautiful interfaces.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js 16.14 or later
- npm (comes with Node.js) or yarn
- React 17 or later
- (Optional) TypeScript 4.5 or later for TypeScript support

## Project Setup

### 1. Initialize Ignix UI

After installing the package, initialize Ignix UI in your project:

```bash
npx @mindfiredigital/ignix-ui init
```

This command will:
- Set up the required configuration files
- Install any peer dependencies
- Configure your project to use Ignix UI components

### 2. Start Using Components

You can now add and import Ignix UI components in your application:

```jsx
npx @mindfiredigital/ignix-ui add component <component-name>
```

## Theme Configuration (Optional)

Ignix UI comes with a default theme, but you can customize it to match your design system. Create a `theme.js` file:

```jsx
// theme.js
export const theme = {
  colors: {
    primary: '#0070f3',
    // Add your custom theme values here
  },
  // Other theme configurations
};
```

Then wrap your application with the `ThemeProvider`:

```jsx
import { ThemeProvider } from '@mindfiredigital/ignix-ui';
import { theme } from './theme';

function App() {
  return (
    <ThemeProvider theme={theme}>
      {/* Your app components */}
    </ThemeProvider>
  );
}
```
